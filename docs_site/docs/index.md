# dsm-tools

Literature-grounded Claude skills, decision engines, and tested templates for
**digital soil mapping**. Location in; an integrated, defensible mapping approach
out - so researchers and policymakers do not have to reason through feature
engineering, model choice, and validation design from scratch.

See the [README](https://github.com/Odunayo3/dsm-tools) for installation, and the
individual skill pages for the judgment each one encodes. Policymakers who only
want to browse (not run) the reasoning can read the skill pages directly.
